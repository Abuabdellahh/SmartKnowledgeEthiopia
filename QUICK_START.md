# Quick Start Guide - Smart Knowledge Ethiopia

## 🚀 Getting Started

### 1. Sign Up

1. Navigate to `/signup`
2. Choose your role:
   - **Student**: For learners and researchers
   - **Teacher**: For educators with moderation access
3. Enter your email and password
4. You'll be redirected to your dashboard

### 2. Dashboard Overview

After signing in, you'll see:
- **Stats**: Books read, AI queries used, voice minutes, notes created
- **Features**: Available tools based on your role
- **Quick Actions**: Direct links to key features
- **Recent Books**: Latest additions to the library

### 3. Using AI Features

#### AI Chat (`/chat`)
- Ask questions about any topic
- Get AI-powered answers
- Context-aware responses
- Available for: Students, Teachers, Admins

#### AI Sidebar
- Appears on book and reading pages
- Tools available:
  - **Chat**: Ask questions about current page
  - **Explain**: Get detailed explanations
  - **Summary**: Quick summaries
  - **Questions**: Generate study questions
- Click the voice button to hear AI responses

#### Voice Reading
- Click the 🔊 button on any book page
- Controls:
  - Play/Pause
  - Speed adjustment (0.5x - 2.0x)
  - Stop
- Available for: Students, Teachers, Admins

### 4. Reading Assistant (`/reading`)

Upload documents for AI-powered analysis:

1. Click "Upload a document"
2. Select a PDF or TXT file
3. Click "Upload & Process"
4. Navigate through pages
5. Use AI tools:
   - Summary
   - Explanation
   - Questions
   - Study Notes
   - Translate to Amharic

### 5. Browse Books (`/books`)

- Search by title, author, or topic
- Filter by category
- View book details
- Read reviews
- Download books (if available)

### 6. Admin Features (`/admin`)

For admin users only:
- Manage books (add, edit, delete)
- View analytics
- Moderate reviews
- System status monitoring

## 💡 Tips & Tricks

### Voice Reading
- Adjust speed for better comprehension
- Use headphones for better audio quality
- Pause and resume anytime
- Works offline once page is loaded

### AI Chat
- Be specific in your questions
- Provide context for better answers
- Use follow-up questions
- Reference specific books or topics

### Reading Assistant
- Upload study materials for analysis
- Use "Questions" tool to create quizzes
- Translate complex texts to Amharic
- Generate study notes automatically

### Search
- Use semantic search for better results
- Search understands meaning, not just keywords
- Try different phrasings
- Filter by category for focused results

## 🆘 Troubleshooting

### Can't access AI features?
- Check your role (must be Student, Teacher, or Admin)
- Ensure you're signed in
- Refresh the page

### Voice reading not working?
- Check browser compatibility (Chrome, Edge, Safari recommended)
- Enable audio permissions
- Try a different browser
- Check volume settings

### Upload failing?
- Check file size (max depends on your Supabase plan)
- Ensure file is PDF or TXT
- Try a smaller file
- Check internet connection
