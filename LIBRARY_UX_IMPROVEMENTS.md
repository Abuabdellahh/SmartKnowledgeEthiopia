3. **Feedback** - Visual response to all interactions
4. **Efficiency** - Quick access to common actions

## 🎨 Color System

- **Primary** - Main brand color for CTAs and accents
- **Secondary** - Supporting color for badges
- **Muted** - Backgrounds and subtle elements
- **Destructive** - Delete actions
- **Foreground** - Text colors
- **Border** - Dividers and card borders

## 📱 Responsive Breakpoints

- **Mobile** - Single column, stacked layout
- **sm (640px)** - 2 columns
- **lg (1024px)** - 3 columns
- **xl (1280px)** - 4 columns

- Card hover: lift + shadow

- Search: instant filtering
- View toggle: background change
- Delete: confirmation dialog
- Transitions: smooth 200-300ms

- Loading state with skeletons
